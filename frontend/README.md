# NeuraScan — Brain Tumor MRI Classifier Frontend

React frontend for the Brain Tumor MRI Classifier.

## Setup
1. Build: `npm install && npm run build`
2. Upload contents of `dist/` folder to this Space
3. Set `VITE_API_URL` to your FastAPI backend Space URL before building
